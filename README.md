## 技术栈

Electron + Vue 3 + TypeScript + Vite + better-sqlite3

## 简介

nucssist 是一款专为安全研究人员和企业安全运维管理打造的桌面级 POC 管理工具。作为 Nuclei Template 管理工具，打通 POC 编写、复现、调试、调度管理的多维度场景，抛弃传统繁琐文件操作，以数据为核心。深度融合 Nuclei Template 语法和 YAML 语法，支持代码高亮、自动提示和语法检查，优化 POC 编写体验。智能识别应用和漏洞类型，提供便捷的应用管理、POC 管理、漏洞类型管理、网空数据管理和任务管理。提供丝滑的实时响应搜索体验。

## 特点

现代化构建方法，更加良好的使用体验，统一、便捷、高效、多变

- **终极 POC 管理** - 编写、归档、查询、调试一站式体验
- **统一 App 管理** - 标准化应用与漏洞类型分类
- **轻量级 Scan 任务管理** - 可视化批量扫描
- **蝇量级 重发器Resender** - 专业 HTTP 请求重放调试
- **重量级网空数据管理** - FOFA/Hunter/Quake 一站式集成

## 特性

1. 智能识别应用和漏洞类型，POC 自动关联应用和漏洞类型
2. 硬派 POC 编写风格
3. 集成 Nuclei Template 语法和 YAML 语法，支持代码高亮、自动提示和语法检查以及代码格式化
4. 提供便捷的应用管理、POC 管理、漏洞类型管理、网空数据管理和任务管理
5. 内置丰富的 Nuclei 语法提示
6. 内置常见安全测试验证字符串
7. 严格的 POC 分级管理和漏洞曝光管理
8. 完全脱离文件，围绕数据操作，更加畅快的操控体验

## 前置条件

### 扫描任务功能依赖

> **重要**：扫描任务功能依赖 Nuclei 工具，使用前请确保：
> 1. 已安装 [Nuclei](https://github.com/projectdiscovery/nuclei) 工具
> 2. 将 nuclei 可执行文件所在目录添加到系统环境变量 PATH

如未正确配置 nuclei 环境变量，扫描任务功能将无法正常使用。

## 快速开始

### 方式一：新建 POC

适用于从零开始编写新的 POC：

1. 下载并安装 nucssist
2. 启动应用，进入「POC 编写」页面
3. 在左侧输入漏洞名称，选择关联的应用（APP），**点击右侧新建按钮**快速创建 POC 模板
4. 在编辑器中调整 template 内容
5. 选择漏洞曝光等级（0day/1day/Nday）
6. 点击「保存」将 POC 存入数据库

### 方式二：导入已有 POC

适用于已有现成的 Nuclei POC 文件：

1. 启动应用，进入「POC 管理」页面
2. 点击「批量导入」按钮
3. 选择包含 POC 文件的目录（支持递归扫描子目录）
4. 配置导入选项：选择重复策略（跳过已存在/覆盖更新）、选择默认曝光等级
5. 点击「开始导入」，等待导入完成
6. 导入的 POC 将自动解析并存入数据库

> **提示**：导入时会自动解析 POC 中的 info 字段，提取应用名称、漏洞类型、严重级别等信息。
